Inspired by https://github.com/joch/wordpress-docker

## Getting started
```
$ git clone --recursive https://github.com/gappc/wordpress-docker.git
$ cd wordpress-docker
$ cp wp-config.php wordpress
$ docker-compose up
```
Browse to http://localhost:8000 and enjoy
